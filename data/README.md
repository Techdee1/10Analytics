# Data Directory

This folder contains the fiscal dataset for the DebtSage project.

## Files

- `data.csv` - Main fiscal dataset with 27 indicators across 14 African countries (1960-2025)
- `problem_statement.csv` - Hackathon problem statement and context
- `10Alytics Hackathon- Fiscal Data.xlsx` - Original Excel file

## Dataset Overview

**Countries (14)**: Egypt, Ethiopia, Ghana, Ivory Coast, Nigeria, South Africa, Tanzania, Togo, Senegal, Kenya, Rwanda, Algeria, Angola, Botswana

**Indicators (27)**:
- Budget Deficit/Surplus
- Capital Expenditure
- Consumer Price Index (CPI)
- Defence Expenditure
- Education Expenditure
- Expenditure
- Exports
- Food Inflation
- GDP Growth Rate
- GDP per Capita
- Government Debt
- Health Expenditure
- Imports
- Inflation Rate
- Interest Rate
- Labour Force
- Nominal GDP
- Population
- Real GDP
- Revenue
- Tax Revenue
- Unemployment Rate
- Value Added Tax (VAT)

**Time Range**: 1960-2025

**Total Observations**: 23,784
